# 🌐 Network Monitor - فاحص الاتصال بالإنترنت

برنامج لمراقبة الشبكة والإنترنت باستخدام بايثون، يقوم بفحص الاتصال بشكل دوري وتسجيل النتائج في ملف Excel مع الوقت والتاريخ.

---

## ✨ الميزات

- ✅ فحص الاتصال بالإنترنت والراوتر
- 📊 تسجيل زمن الاستجابة (Ping) بالمللي ثانية
- 📁 حفظ السجلات في ملف Excel مع التاريخ والوقت
- 🖥️ واجهة مستخدم بسيطة وسهلة (Tkinter)
- 🔄 فحص تلقائي كل 5 ثواني
- 📋 عرض جميع السجلات داخل البرنامج

---

## 🛠️ التقنيات المستخدمة

- **Python 3.14+**
- **Tkinter** – واجهة المستخدم
- **Pandas** – التعامل مع ملفات Excel
- **Subprocess** – تنفيذ أوامر النظام (Ping)
- **Threading** – تشغيل الفحص في الخلفية
- **PyInstaller** – تحويل البرنامج إلى ملف EXE

---

## 📁 هيكل المشروع

```
network-monitor/
├── network_monitor.py      # الكود الأساسي
├── requirements.txt        # المكتبات المطلوبة
├── network_log.xlsx        # ملف السجلات (يتولد تلقائياً)
└── README.md               # شرح المشروع
```

---

## ▶️ طريقة التشغيل

1. تأكدي من تثبيت **Python** على جهازك.
2. حمّلي المكتبات المطلوبة:

```bash
pip install -r requirements.txt
```

3. شغّلي البرنامج:

```bash
python network_monitor.py
```

---

## 🛠️ تحويل البرنامج إلى EXE

لتوزيع البرنامج بدون الحاجة إلى تثبيت Python:

```bash
pip install pyinstaller
pyinstaller --onefile --windowed network_monitor.py
```

بعد الانتهاء، ستجد ملف `network_monitor.exe` في مجلد `dist`.
